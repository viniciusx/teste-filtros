Esta pasta contém os resultados da comparação de similaridade entre aquivos de áudio puros (simplesmente baixados da internet, sem nenhum tipo de edição) e arquivos melhorados através do uso das ferramentas citadas aqui: 
https://github.com/viniciusx/teste-filtros/wiki/Dados.
O valor do resultado representa a porcentagem de similaridade dos dois textos (transcrição automatizada e manual).

Em relação ao nome dos arquivos:
	- Arquivos *.txt se referem a testes feitos com áudio puro.
	- Arquivos *_ed.txt se referem a testes usando primeiro Normalizador e depois nivelador.
	- Arquivos *NOTRE.txt e *RE.txt se referem a testes que utilizaram o plugin Vocal Remover, onde foi removido frequências acima de 3500Hz e menores que 250 Hz (voz humana está entre esses dois).
	- Arquivos *_levelnorm se referem a testes usando primeiro o Nivelador e depois Normalizador.
	- Arquivos *_level se referem a testes usando apenas o Nivelador.
	- Arquivos *_norm se referem a testes usando apenas o Normalizador.
	- Arquivos *_lowpass se referem a testes usando apenas o Low Pass.
	- Arquivos *_lowpass se referem a testes usando apenas o High Pass.
	- Arquivos *_lowpass se referem a testes usando High Pass e Low Pass.
